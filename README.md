
Dynamic Size Pricing

Size should matter! With this mod, scaling or resizing Build/Buy objects adjusts the price to match: a §800 sofa scaled to 150% costs §1,200, shrink it and you pay less. You're charged or refunded the size premium the moment you confirm a new size and selling a modified item refunds the full amount you paid, premium included. Catalog prices stay untouched and vanilla.

Works on any scalable or resizable catalog object (furniture, decor, windows…). Walls, floors, and terrain are unaffected. Requires BepInEx 5.
